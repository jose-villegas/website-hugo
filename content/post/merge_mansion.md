I moved to Berlin to join [Metacore](https://metacoregames.com/) as a Senior Game Developer, working on Merge Mansion, a merge-and-story puzzle game with a massive live player base and years of established codebase and content pipelines.
<!--more-->

My work here has focused on contributing to new live events, tools and test framework for the game, optimizing game performance and modernizing the existing codebase with new programming patterns. This included a self-balancing random weights system, which algorithmically adapts random weights depending on conditions or context, and a tool to browse the item structure using a treemap perspective, similar to WinDirStat. I've also been developing AI-assisted engineering practices, using LLMs for architecture work, context harnessing and feature development.

The game can be found on [Apple Store](https://apps.apple.com/us/app/merge-mansion-puzzles-story/id1484442152) and [Google Play](https://play.google.com/store/apps/details?id=com.everywear.game5).
